4. WIDGETS DE EJERCICIOS - FUNCIONALIDAD
================================================================================

4.1 EJERCICIO DE ESCRITURA
- Input de texto libre
- Validación NO case sensitive (configurable por ejercicio)
- Ignorar acentos (configurable, default: true)
- Ignorar puntuación (configurable, default: true)
- Tolerancia de errores: distancia Levenshtein configurable (default: 0)
- Feedback visual inmediato o al enviar (configurable)
- Algoritmo de normalización:
  1. trim()
  2. toLowerCase() si !case_sensitive
  3. normalize("NFD").replace(/[\u0300-\u036f]/g, "") si ignorar_acentos
  4. replace(/[.,\/#!$%\^&\*;:{}=\-_`~()?"']/g, "") si ignorar_puntuacion
  5. Comparar con tolerancia Levenshtein

4.2 EJERCICIO DRAG AND DROP - TIPOS
- Ordenar: ordenar palabras/frases para formar oración correcta
- Emparejar: columna A con columna B (vocabulario)
- Completar: arrastrar opciones a espacios en blanco
- Soporta texto e imágenes
- Biblioteca recomendada: SortableJS o implementación nativa HTML5 Drag API

4.3 EJERCICIO LISTENING (TTS)
- Usar Web Speech API del navegador (Chrome requerido)
- Texto a leer desde contenido JSON
- Velocidad configurable (0.5x a 2.0x)
- Repeticiones permitidas configurables
- Botones: reproducir, pausar, velocidad
- Fallback si navegador no soporta TTS

4.4 EJERCICIO MICRÓFONO (STT + Grabación)
- Modo seleccionable por ejercicio:
  a) Grabación solo: guarda audio para que profesor escuche después
  b) Evaluación automática: Web Speech API transcribe y compara
- Idioma STT configurable: 'en-US' (inglés), 'fr-FR' (francés)
- Guardar audio: configurable (default: true)
- Días de retención: 7 días después de finalizar curso (cron job de limpieza)
- Comparación automática usa misma lógica de tolerancia que escritura
- Feedback: transcripción reconocida, porcentaje de coincidencia
